# Demoblaze

> Pruebas automatizadas para el sitio Product Store, estas pruebas cubren los principales flujos del negicio.

## Tabla de Contenidos
- [Descripción] Flujos principales del sitio Product Store
- [Características] Flujso automatizadas
- [Tecnologías-Utilizadas] Cypress 15v
- [Uso] Ejercicio de automatizacion
- [Autor] Kirvan Suarez 

---

## Descripción
Al ejecutar el set de pruebas automatizadas resuelve principalmente la ineficiencia, el error humano y la lentitud del control de calidad manual, permitiendo validar rápidamente la funcionalidad y estabilidad del sitio de los flujos principales.

## Características
- **Característica 1:** Inicio de sesión
- **Característica 2:** Agregar producto a carrito 
- **Característica 3:** Ver producto agregado en carrito
- **Característica 4:** Solicitud de compra de producto finalizada

## Tecnologías Utilizadas
- **Lenguaje:** JavaScript
- **Framework:** Cypress

## Prerrequisitos
Qué necesitas tener instalado antes de comenzar:
- Node.js (v16+)
- npm (v10+)
- Cypress (v15+)
- Git
